# Agents

Instructions for choosing the right agent for each task.

## Available Agents

### build
Use for: writing code, creating files, implementing features, fixing bugs.
Best for tasks that require making changes to the codebase.

### plan
Use for: analyzing problems, planning architecture, researching options.
Best for tasks that require thinking before acting.

## Guidelines
- Default to `build` for most tasks
- Use `plan` when the user asks for analysis or when the task is complex and unclear
- Always confirm with the user before making large irreversible changes
