# Skill: Daily Summary

Generate a concise summary of the day's activity and pending tasks.

## When to use
- Triggered by the daily cron job (default: 8:00 AM)
- When the user asks "what's on my agenda", "summarize today"

## Instructions
1. Read `memory/memory.md` for recent notes and tasks
2. Read `memory/context.md` for the current project focus
3. Generate a structured summary with:
   - Current project/focus
   - Pending tasks or reminders
   - Any important notes from memory
4. Keep the summary brief — 5 to 10 bullet points maximum
5. End with a motivating closing line
