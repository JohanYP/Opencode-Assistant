# Skill: Web Search

Search for up-to-date information on the internet.

## When to use
- The user asks for current news, prices, or data
- You need information that may have changed since your training cutoff
- The user asks "search for...", "find...", "look up..."

## Instructions
1. Use the available web search tools to find relevant information
2. Summarize the findings clearly and concisely
3. Always cite sources when providing factual information
4. If multiple results are found, present the most relevant ones
